## Dataset

The dataset used for training and testing is too large to be stored directly in this repository.  
You can download it from the following link:

[Download Dataset](https://drive.google.com/drive/folders/1_Bp-PCc1kBPTmEuaXThAx-qNVcGEr7p_?usp=drive_link)

After downloading, place the dataset inside the `dataset/` folder in the project root before running the application.

## Pre-trained Model
Download the trained CNN model here: [plant_model.h5](https://drive.google.com/file/d/1BBxftUqFlkrECALN0N-l_DcdVx1TGAiY/view?usp=drive_link)

Place the file inside the `model/` folder before running the app.
